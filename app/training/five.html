Javascript

JS הייתה מאז ומתמיד שפה חשובה, הלא זו השפה שרצה בדפדפן! אך כיום Javascript תופסת את מעמדה כשפה מובילה בפיתוח צד שרת, אפליקציות מובייל, משחקים, רובוטים, ומקומה בדברי הימים הטכנולוגיים מובטח!



AngularJS

הFramework האולטימטיבי מבית Google שכבש בשערה את השוק, החל מהסטרטאפים הכי חדשים וכלה באפליקציות אנטרפרייז בנקאיות.

Open source, two way data binding, dependency injection , single page routing, testable components

Sass, gulp, browserify, npm, bower yeoman, Jasmin, karma, protractor, Bootstrap…
